HeartCom

Sistema de Gestión para Junta de Vecinos | 2025 – 2026

Desarrollo completo de un sistema web desde cero, diseñado para administrar toda la información, socios y actividades de una junta de vecinos. Cuenta con altos estándares de seguridad y facilidad de uso.

  • Seguridad: Autenticación segura con doble factor (MFA) y recuperación de contraseña.
  • Gestión: Control de roles, permisos, sesiones y datos de usuarios.
  • Módulos: Administración de socios, registro de pagos, reservas de espacios y comunicados.
  • Tecnologías: PHP, MySQL, Bootstrap, JavaScript, AJAX, jQuery.
  • Base de datos: Diseño relacional normalizado y optimizado para consultas rápidas.

Módulo de Pagos

Aplicación de Transporte de Buses | 2025 – 2026

Desarrollo del sistema de pagos y carga de tarjetas para una aplicación móvil y web de transporte público. Integración directa con la pasarela de pagos estándar en Chile.

  • Integración: Conexión completa con WebPay para procesar transacciones seguras.
  • Funcionalidad: Carga de saldo, gestión de tarjetas asociadas y historial de movimientos.
  • Seguridad: Validación estricta de datos, cifrado de información y control de estados de pago.
  • Arquitectura: Programación backend robusta orientada a la protección de datos sensibles.
  • Tecnologías: PHP, MySQL, WebPay API.
